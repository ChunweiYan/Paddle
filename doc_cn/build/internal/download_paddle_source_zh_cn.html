下载PaddlePaddle的源码¶

PaddlePaddle的源码保存在公司的SVN服务器上。

下载PaddlePaddle代码可以从任意路径下载。

使用SVN下载PaddlePaddle源代码¶

PaddlePaddle的SVN路径为 https://svn.baidu.com/idl/trunk/paddle, 下载该源码使用命令

svn co https://svn.baidu.com/idl/trunk/paddle

即可下载源码。

使用Git下载PaddlePaddle源代码¶

PaddlePaddle的git mirror地址为 https://git.baidu.com/idl/paddle.git，下载该源代码使用命令

git clone https://git.baidu.com/idl/paddle.git

即可下载源代码。

What’s Next?¶
