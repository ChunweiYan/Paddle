下载PaddlePaddle的源码
==================

PaddlePaddle的源码保存在公司的SVN服务器上。

下载PaddlePaddle代码可以从任意路径下载。


使用SVN下载PaddlePaddle源代码
```````````````````````````

PaddlePaddle的SVN路径为 `https://svn.baidu.com/idl/trunk/paddle <https://svn.baidu.com/idl/trunk/paddle>`_, 下载该源码使用命令


..  code-block:: bash
    
    svn co https://svn.baidu.com/idl/trunk/paddle

即可下载源码。

使用Git下载PaddlePaddle源代码
``````````````````````````


PaddlePaddle的git mirror地址为 `https://git.baidu.com/idl/paddle.git <https://git.baidu.com/idl/paddle.git>`_，下载该源代码使用命令

..  code-block:: bash

    git clone https://git.baidu.com/idl/paddle.git

即可下载源代码。
